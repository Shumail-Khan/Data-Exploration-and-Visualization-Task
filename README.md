# Task 1: Exploring and Visualizing a Simple Dataset
#### Objective:
Learn how to load, inspect, and visualize a dataset to understand data trends and distributions.
Dataset:
Iris Dataset (CSV format, can be loaded via seaborn or downloaded)

#### 
Instructions:
<ul>
<li>Load the dataset using pandas.</li>
<li>Print the shape, column names, and the first few rows using .head().</li>
<li>Use .info() and .describe() for summary statistics.</li>
</ul>
Visualize the dataset:
<ul>
  <li>Create a scatter plot to show relationships between features.</li>

  <li>Use histograms to show value distributions.</li>
  <li>Use box plots to identify outliers.</li>
  <li>Use matplotlib and seaborn for plotting.</li>
</ul>
Skills:
<ul>
<li>Data loading and inspection using pandas</li>
<li>Descriptive statistics and data exploration</li>
<li>Basic plotting and visualization with seaborn and matplotlib</li>
</ul>
